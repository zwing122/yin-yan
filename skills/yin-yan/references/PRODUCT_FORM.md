# 产品形态

## 第一技术存在形式

第一版是 OpenClaw/Hermes/QClaw 中的个人财务助理 skill pack。

它不是独立 App，不是完整 SaaS，也不是自动记账工具。

## 宿主平台提供

- 用户已有聊天或 IM 入口。
- agent 运行时。
- skill 加载机制。
- heartbeat、cron 或定时任务触发能力。
- 本地文件读写环境。
- 用户自己的模型配置或 API key。

## Skill 自己承担

- `SKILL.md` 行为入口。
- 自然语言记账、查询、复盘、备份恢复脚本。
- 本地 `ledger.json` 读写契约。
- 高歧义事件复核规则。
- 心跳摘要模板。
- 匿名 `test_report` 生成契约。
- 本地网页导入和图表复盘契约。

## 不承担

- 云托管 OpenClaw/Hermes/QClaw。
- 托管用户 IM 凭证。
- 自动读取微信、支付宝或银行流水。
- 自动支付、转账、催收或投资执行。
- 完整账号体系和订阅计费系统。
- 服务端保存完整账本。

## 长期产品形态

可以演化为混合形态：

- OpenClaw/Hermes/QClaw skill pack。
- 本地账本内核。
- 本地网页复盘器。
- 可选托管解析接口。
- 后续创业者版 API、SDK、规则包或私有化模块。

## 验证顺序

1. 固定本地账本和匿名报告 schema。
2. 实现最小 skill 行为。
3. 用小范围内测跑 7 天。
4. 收匿名报告和脱敏错误样例。
5. 再决定是否扩大本地网页复盘器或托管解析接口。
